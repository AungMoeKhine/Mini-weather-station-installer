ESP32 Mini Weather Station Web Installer

This program was developed by Aung Moe Khine.

Connect your ESP32 to your computer via USB, then use Chrome or Edge and click the button below to install the firmware. Download the required icon folders, place them in the SD card's root directory, and insert the SD card into the slot. Without the SD card and icons, will not fetching weather data and display the Weather icons.

Please use ESP32 CYD 2.8 inches board only
